## 💡 About the Project

**PCIA** is an experimental **autonomous desktop research agent** designed to bridge the gap between **human intent** and **system execution**.

Unlike traditional automation tools, PCIA combines:

- 🧠 **Chain-of-Thought planning**
- 👁️ **Visual verification**
- ⚙️ **Real-time system observation**

to perform **complex web research and automation workflows** directly on Windows.

## 🧠 Core Capabilities

### 🔹 Cognitive Planning
Uses **`gemini-3.1-flash`** to generate structured **JSON task chains** that guide the execution pipeline.

---

### 🔹 Recursive Mini-Brain
When dynamic UI appears (like Google search results), PCIA spawns a **tactical Mini-Brain** to analyze screenshots and perform precise actions.

---

### 🔹 Self-Healing Execution
Execution errors are automatically recorded in:
`learning_log.json`

These logs are injected into the planner to **avoid repeating failures**.

---

### 🔹 Physical Scraping
Instead of APIs, PCIA uses:
`
Ctrl + A
Ctrl + C
`

to ingest webpage content directly from the browser.

This bypasses most **anti-bot scraping protections**.

---

### 🔹 Context Awareness
The agent monitors:

- OS
- CPU load
- RAM usage

to dynamically adjust execution timing and stability.
